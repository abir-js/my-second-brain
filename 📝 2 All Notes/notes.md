# NextJS
## What is NextJS

- NextJS is a fullstack framework build on top of react.
- SSR(server-side rendering), routing, optimisation
- developer experience
- 
## NextJS vs only react

| topics                    | React JS         | Next JS            |
| ------------------------- | ---------------- | ------------------ |
| **routing**               | Manual Setup     | Build In           |
| **server-side rendering** | Manual Config    | Build In support   |
| <br>**optimization**      | Manual           | optimisation       |
| **SEO** <br>              | Limited(CSR)     | Automatic(SSR)     |
| **API Route**             | Seperate Backend | Build in API route |

## Routers in NextJS

- page router (old)

- app router (new)
