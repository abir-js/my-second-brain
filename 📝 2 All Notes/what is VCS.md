# What is VCS?

- VCS means **Version Controlling System**
- Git is also VCS.
- There are multiple VCS like Mercurial, as well as older centralized systems such as Subversion (SVN), CVS, and Perforce

VCS Remote:
	1. Github
	2. Gitlab
	3. GitTea
	4. AWS CodeCommit
