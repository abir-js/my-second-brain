# What is Interface?

- Interface is a **blueprint** of a class.

e.g.

- Car ( wheel, speed, engine ) - Interface
- Maruti 800 - class
	- Maruti 800 lower model - object
	- Maruti 800 middle model - object
	- Maruti 800 higher model - object
