## Singly LinkedList

1. **Create/Insert**
	1. At start
	2. middle/position

2. **Deletion**
	1. first

```java
class Node {
	int data;
	Node next;
	
	// Constructor
	Node(int x){
		data = x;   
		next = null 
	}
}

class linkedlist{
	main(){
		Node temp = new Node(10);
	}
```
